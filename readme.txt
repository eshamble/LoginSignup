To run the website, start on register.html.

Emails must follow an email format:
    [a-z0-9._%+-]@[a-z0-9.-].[a-z]{2,}$"

Password and repeat password must have the same input,
otherwise you cannot move on to the next page.

Login does not have those specifications yet.

When you log in, you are led to the menu. The one
button that leads to something currently is "new product".
